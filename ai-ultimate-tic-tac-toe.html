AI Ultimate Tic Tac Toe

02/25/2019

Click here for more information on gaming rules

This program utilizes AI gaming strategy to play an offensive agent and defensive agent against one another in a game of Ultimate Tic Tac Toe. In this, methods were developed to determine the optimal move each agent should play that would yield the highest reward and eventually win the game (or prevent an agent from winning).

Technical Information

Language: Python


How it works

The program uses the Minimax algorithm with alpha beta parameters in order to prune the less optimal potential steps for the offensive or defensive agent. Heuristics are implemented to translate the gain value of each step that could be taken from the current state in order to determine the best move for the agent.
