Restaurant Service Application

04/20/2020

Purpose

The purpose of this project was to develop an application that would improve service efficiency for a restaurant. In turn, the application would allow for greater productivity and flexibility for the restaurant while reducing the amount of resources required to successfully run a food establishment.

Services

Customer

  • May register or login to account to earn rewards
  • View food items, prices, descriptions, and nutritional features
  • Play games
  • Submit payment through the application, including tips
  • Can request service at any time

Waiter

  • Can view table alerts for help and refills in real time
  • Can view table status and kitchen order status
  • Can clock in and clock out
  • Can verify personal information

Kitchen Staff

  • Can view orders sent directly from customer table in real time
  • Can send updates to waiter
  • Can edit menu items
  • Can view and update inventory
  • Can verify personal information

Manager

  • Can oversee anything the staff and customer have access to/ are able to do
  • Can apply discounts
  • Can hire, fire, and update employees
