Software Engineering Intern

07/06/2020

JP Morgan Chase | Chicago, IL

Click here to view more about the nonprofit.

Our Task

NHS is a local Chicago based non profit that seeks to create opportunities for people to live in affordable homes, improve their lives, and strengthen their neighborhoods.

Our challenge was to utilize adaptive technology to help NHS engage and grow their younger audience, while maintaining the core NHS experience for other users.

Project Solution

End product contains confidential information. Because of this, a demo cannot be displayed.

A responsive and mobile friendly web app with an event planner and interactive assistant, that could be easily integrated into the existing NHS site.

Responsibilities

  • Formulated backend algorithm of an automated chatbot feature to improve service usability
  • Developed Django API calls to store and retrieve chatabot data from MySQL database
  • Led team in the steps of design and implementation for the chatbot feature

Technology

  • Frontend: ReactJS | HTML | CSS
  • Backend: Django | Python | MySQL

Our Impact

A more user friendly platform

Our end product consisted of making navigation of NHS resources for a new user easier as all the information is one place with a clear format. Additionally, the platform is more interactive for users as the chatbot is there to help guide users through NHS resources.

A more modifiable platform

Both of the implemented features have associated administrator components that allow NHS workers to add, edit, and remove from their platform with ease without any technical knowledge. This way, the product we have developed will be maintainable and ever evolving with the non profit.
