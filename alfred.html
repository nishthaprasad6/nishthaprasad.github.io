ALFRED

05/11/2020

A Benchmark for Interpreting Grounded Instructions for Everyday Tasks | Visual Language Navigation

The ALFRED project is pivotal in breaking strides for visual language navigation as it is designed to have an agent follow a series of user given instructions in a simulated environment. For more detailed information, please refer to the following research report.

DOWNLOAD Alfre...port.pdf

My Goal

The current dataset and model functions under the assumptions that all instructions are valid in the given environment. This is not always true in reality. I, and a team of three others, are currently working to design and improve on the given dataset. We are doing so by implementing a model to train the agent to know what is possible and what is not.

Please click here to see the repository

Undergraduate AI Research Assistant

  • Conducted research in natural language and machine learning for NSF-funded REU program

  • Constructed dataset and built neural network models for improving visual language navigation

  • Defined and evaluated strategies to automatically generate adversarial examples and make models more robust

What We Have Accomplished

  • Preprocessing our data in which per image there is one correct instruction and one incorrect instruction (that still appears to be logical)
  • Developing a binary classification model prototype that takes two inputs, an image and an instruction, with label that is 0 or 1 (representing True or False)

What We Have Left To Do

  • Optimizing our model hyperparameters to yield greater accuracy
  • Improving upon our input data to be more realistic
  • Determining further steps upon how agent should go about once knowing the instruction is incorrect based off our model
