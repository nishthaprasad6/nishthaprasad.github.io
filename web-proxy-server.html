Web Proxy Server

03/04/2020

This program uses TCP sockets to create a web proxy server in which a single client is able to send HTTP requests to the server. Furthermore, the server is able to cache the five most recently visited websites in order to reduce network traffic and improve efficiency.

Please click here to see the repository

Technical Information

Software: TCP Sockets in C


How it Works

The client connects to the server and requests to read a user input URL. The proxy server first checks the cache if the webpage has been recently visited. The cache is a text file that stores the url as well as the data last accessed. If the URL is not in the cache, a GET request is sent to the web server and the URL is added to the cache. In the end, the client receives the HTML body of the URL it requested.
